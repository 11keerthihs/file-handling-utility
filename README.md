# 📂 File Handling Utility (Java)

## 📌 Internship Task - 1

**Title:** File Handling Utility
**Objective:** Create a Java program to read, write, and modify text files.

---

## 🧾 Project Description

The **File Handling Utility** is a Java-based application developed as part of an internship task.
This program demonstrates core file operations such as **reading**, **writing**, and **modifying** text files using Java's file handling classes.

The project helps in understanding how data is stored, accessed, and manipulated in files, which is essential for real-world software development.

---

## 🎯 Features

* 📖 Read content from a text file
* ✍️ Write data into a text file
* 🔄 Modify (update/append) existing file content
* ❌ Handle file-related errors (e.g., file not found)
* 📁 Simple and user-friendly structure

---

## 🛠️ Technologies Used

* **Java**
* **File Handling Classes**

  * `FileReader`
  * `BufferedReader`
  * `FileWriter`
  * `BufferedWriter`

---

## ⚙️ How It Works

### 1. Reading a File

The program reads data from a text file using `BufferedReader` and displays it on the console.

### 2. Writing to a File

It allows writing new content into a file using `FileWriter`.

### 3. Modifying a File

The program can append or update existing content without deleting previous data.

---

## ▶️ How to Run

1. Compile the Java program:

```bash
javac FileHandlingUtility.java
```

2. Run the program:

```bash
java FileHandlingUtility
```
